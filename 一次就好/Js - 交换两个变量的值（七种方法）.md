### 七种方案解决JavaScript交换两个变量值的问题

var a = 1,b=2;//交换后：a=2;b=1;

1. 临时变量（最简单）

   ```javascript
   var c = a;
   b=a;
   b=c;
   ```

   

2. 加减交换（限定数字）

   ```javascript
   a+=b;
   b=a-b;
   a-=b;
   ```

   

3. 位运算交换（限定数字）

   ```javascript
a^=b;
   b^=a;
   a^=b;
   ```
   

   
4. 对象键值交换

   ```javascript
a = {a:b,b:a};
   b = a.b;
   a = a.a;
   ```
   

   
5. 数组的值交换

   ```javascript
a = [a,b];
   b = a[0];
   a = a[1];
   ```
   

   
6. 一行交换

   ```javascript
a = [b,b=a][0];
   ```
   

   
7. 解构赋值交换

   ```javascript
[a,b] = [b,a];
   ```
